# Conva

```bash
npm install #server packages
npm run client-install #client packages
npm run dev #client & server localhost
```

Backend at port: 5000

Frontend at port: 3000

To run only server

```bash
npm run server
```

To run only client

```bash
npm run client
```

Server Testing

```bash
npm test
```

Client Build

```bash
npm run client-build
```
